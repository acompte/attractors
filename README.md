# Wokshop on attractor networks for simple cognitive tasks

In this repository you can find printouts of the presentations of the course (\*.pdf), together with their souce code in [Quarto](https://quarto.org){:target="_blank"} (\*.qmd).

You can run the exercises in `attractors_part1_notebook.ipynb` directly in [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/acompte/attractors/main?urlpath=%2Fdoc%2Ftree%2Fattractors_part1_notebook.ipynb)  or alternatively setup a local Python environment with the requirements set in `requirements.txt` and run the notebook locally in your favorite system.
